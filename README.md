# Like a Canvas

株式会社Aoi — OEMグッズ制作サービスサイト

## 概要

デザイン〜製造〜納品を一括対応するOEMグッズ制作サービス「Like a Canvas」のウェブサイトです。

## 構成

```
like-a-canvas/
├── index.html          # トップページ
├── pages/
│   ├── factory.html    # 提携工場 (Factory Network)
│   ├── what-we-can-do.html  # 何ができる？ (Service Overview)
│   ├── about.html      # About Us
│   └── contact.html    # お問い合わせ
├── assets/
│   ├── css/main.css    # メインスタイルシート
│   ├── js/main.js      # メインスクリプト
│   ├── images/         # 画像素材
│   └── videos/         # 動画素材
└── data/               # 制作実績データ等
```

## 技術構成

- 静的HTML / CSS / JavaScript（フレームワークなし）
- フォント：Oswald (heading) / Open Sans (body) / Vollkorn (serif)
- フォームバックエンド：Formspree

## 開発メモ

- 画像・動画ファイル名の整理は次フェーズで実施予定
- 独自ドメインでの公開は別ステップで対応
