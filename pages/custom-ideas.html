Custom Ideas

Custom
Ideas

グッズは、ロゴを載せるだけでなく、素材や加工、ディテールによって印象が大きく変わります。ここでは、これまでの実績をもとにしたカスタムのアイデアをご紹介します。

Filter

うちわにタッセルを付ける

コンサートグッズの定番・うちわに房飾り（タッセル）を追加。単価を抑えながら、手元の印象を大きく変えるディテールワーク。

Tassel Detail Event Goods

反射プリントを入れる

ライブやフェスの照明に反応するリフレクティブプリント。通常のシルクスクリーンと組み合わせることで昼夜で異なる表情をつくる。

Reflective Print Silkscreen

バッグをパッカブル仕様にする

収納ポケットに本体をまるごと収納できるパッカブル設計。軽量ナイロンと組み合わせ、フェス・アウトドアシーンに合わせた実用性を持たせる。

Packable Nylon Functional

キャップにトリコロールのネームを付ける

フランスのカフェをテーマにしたキャップに、青・白・赤のトリコロール配色のオリジナルネームテープを縫い付け。世界観を細部で表現する。

Custom Tag Embroidery World-building

染め・洗い加工で風合いを出す

ガーメントダイ（製品染め）や洗い加工で、新品にヴィンテージライクな風合いを与える。同じシルエットでも素材処理で別物の印象になる。

Garment Dye Washed Original Fabric

Tシャツの生地・厚み・シルエットから考える

既製品では出せない重量感・シルエット・質感を、生地の仕様から設計する。320gのヘビーウェイトや、ドロップショルダーなど体型カバーのカット設計も可能。

Original Fabric Heavy Weight Full Order

タグ・ネーム・ピスネームで世界観を作る

下げ札・ネームテープ・ピスネーム（ワッペン）などの付属品を統一することで、グッズ全体のブランド感が大きく上がる。印刷から縫い付けまで一括対応。

Custom Tag Hang Tag Branding

撥水・防水・メッシュ・リップストップ素材を使う

機能素材を採用することで、アウトドア・フェスシーンに実用的なアイテムに仕上がる。軽量リップストップナイロンや撥水加工コーティングも選択可能。

Water-repellent Ripstop Functional

オリジナルパッケージでスパイスを加える

OPP袋・紙箱・巾着など、商品を包む資材もオリジナルで制作。グッズを開封するシーンまで設計することで、受け取り体験を演出する。
